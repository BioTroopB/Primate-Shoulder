## AI Extension ‚Äî PrimateOsteoID.ai

This M.A. dataset has been extended into **PrimateOsteoID.ai**, a fully automated, open-source AI classifier that predicts **species, sex, and side** from raw 3D shoulder bone scans ‚Äî **no manual landmarking required**.

**Live Demo (v1 ‚Äî stable public version)**
